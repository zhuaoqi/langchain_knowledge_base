# Langchain Knowledge Base

这是一个基于 LangChain 的知识库项目，包含以下内容：

- 文档处理脚本
- 向量数据库配置
- 知识检索工具
- 对话系统实现

## 项目结构

主要目录说明：

- `data/`: 存放原始文档数据
- `db/`: 数据库相关脚本
- `models/`: 模型文件
- `utils/`: 工具脚本
- `volumes/`: 数据库存储卷
